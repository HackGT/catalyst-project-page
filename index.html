Now that we know most of the basics of HTML, we're going to try and start making a project page to showcase what we've made at Catalyst

A few more helpful HTML elements we'll learn are the 'a' tag and the 'img' tag

To make links in HTML, we use the 'a' tag.

You can link to sites like this!

The 'href' attribute in the 'a' tag represents the destination we're trying to go to

Make sure you add the 'http://' to the front of your link, otherwise it'll act funny

The 'img' tag lets us embed imaged

We put the image url in the 'src' attribute, not an 'href' attribute

Also, notice that the 'img' tag doesn't have a closing tag

This is because images don't really have anything inside of them
